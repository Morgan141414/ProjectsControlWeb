# ProjectsControl Desktop (PySide6)

Native desktop client for Windows/macOS.

## Setup

```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Run

```
python -m app.main
```

## Notes

- API base URL is set in `app/services/api_client.py`.
- Start the backend before running the app.
